Digital Goods API is mainly focused on selling In-game virtual items, services and premium membership with PayAlto.

Stored widget call

It shows a widget, where stored products in Products section of Merchant Account are available for payment. Both subscription and one-time payment are supported in our products section.

Library:
https://github.com/paymentwall/paymentwall-java
Request URL
https://api.paymentwall.com/api/subscription/
Request method
https://api.paymentwall.com/api/subscription/
Initialize
              
import com.paymentwall.java.Base;

Base.setAppKey("YOUR_APPLICATION_KEY");
Base.setSecretKey("YOUR_SECRET_KEY");
Base.setApiType(Base.API_GOODS);

Parameter:

Name Required Format Description
key yes String(32) Sample: 048c51f8fc834a1467db96b683f16e70 Can be found in General Settings of the Project inside of your Merchant Account
uld yes String(64) Sample: test@paymentwall.com Not case-sensitive. ID of the end-user in your system (usually it is an email) who is viewing the widget. The maximum length is 64.
widget yes String Sample: p1_1 Widget code. Can be obtained by Merchant Account -> Projects -> Widgets
email yes String Sample: test@paymentwall.com Paymentwall will automatically send a payment receipt to the user.
timestamp yes String(10) Sample: 1477985778 Unix timestamp of the current date
sign_version no String Sample: 2, 3 Signature Version.
sign no String Sample: 7633c3b060fa2ddd41283ff d4629ab40 Refer to Signature Calculation .
Stored Widget Call
                        
import com.paymentwall.java.Widget;
import com.paymentwall.java.WidgetBuilder;
import java.util.LinkedHashMap;

WidgetBuilder widgetBuilder = new WidgetBuilder(userid, widgetcode);

widgetBuilder.setExtraParams(new LinkedHashMap <String, String>(){
    {
        put("email", email);
        put("timestamp", timestamp);
    }
});

Widget widget = widgetBuilder.build();
